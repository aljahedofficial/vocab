## **PHASE 1: MVP - CORE FEATURES (Weeks 2-3)**

### **Task 1.1: Backend - User Authentication**
- [ ] Create `/users` table (user_id, email, password_hash, created_at)
- [ ] Implement user signup endpoint
- [ ] Implement user login endpoint
- [ ] Add JWT token generation
- [ ] Add middleware for protected routes

**Technology:** Node.js + Express (or Python + FastAPI)

**Key Endpoints:**
```
POST   /api/auth/signup
POST   /api/auth/login
POST   /api/auth/logout
```

**Deliverable:** Users can register and log in

---

### **Task 1.2: Backend - Document Upload & Storage**
- [ ] Create `/documents` table (document_id, user_id, filename, file_type, upload_date, etc.)
- [ ] Implement file upload endpoint
- [ ] Store files in **Supabase Storage**
- [ ] Validate file types (pdf, docx, txt only)
- [ ] Validate file size (<50MB)

**Key Endpoints:**
```
POST   /api/documents/upload
GET    /api/documents/list
DELETE /api/documents/:id
```

**Libraries:** multer (Node) or werkzeug (Python)

**Deliverable:** Users can upload files, files stored securely

---

### **Task 1.3: Backend - Text Extraction**
- [ ] Install text extraction libraries:
  - pdfplumber (PDF)

- [ ] Create text extraction function
- [ ] Handle encoding detection (UTF-8, Bengali scripts)
- [ ] Create `/extract` endpoint

**Key Endpoint:**
```
POST   /api/documents/:id/extract
```

**Deliverable:** Extracted text from all file types

**Key Endpoint:**
```

```


**Deliverable:** Word frequencies calculated and stored
